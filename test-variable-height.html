How It Works

When variableHeight="true" is set on a vertical split:

  1. Dynamic Height: The split component's height adjusts automatically (height: auto)
  2. Special Drag Behavior: Dragging a gutter only resizes the area above it
  3. Fixed Areas Below: All areas below the dragged gutter maintain their original sizes
  4. Component Growth: The entire component grows or shrinks based on the top area's size

Usage Example

<as-split
  direction="vertical"
  unit="pixel"
  [variableHeight]="true"
  [gutterSize]="11"
>
  <as-split-area [size]="150" [minSize]="50" [maxSize]="300">
    <!-- This area changes size when dragging -->
  </as-split-area>
  
  <as-split-area [size]="100">
    <!-- This area keeps its size -->
  </as-split-area>
</as-split>
